## 🎯 核心功能清单

### 后端API接口

#### 认证服务（auth-service）
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/register/email` - 邮箱注册
- `POST /api/auth/login/username` - 用户名登录
- `POST /api/auth/login/phone` - 手机号登录
- `POST /api/auth/login/phone-code` - 验证码登录
- `POST /api/auth/login/wechat` - 微信登录
- `POST /api/auth/forget-password` - 忘记密码
- `POST /api/auth/reset-password` - 重置密码
- `POST /api/auth/verify-code/email` - 发送邮箱验证码

#### 内容服务（content-service）
- `GET /api/content/article` - 获取文章列表
- `GET /api/content/article/{id}` - 获取文章详情
- `POST /api/content/article` - 创建文章
- `PUT /api/content/article/{id}` - 更新文章
- `POST /api/content/article/{id}/publish` - 发布文章
- `POST /api/content/article/{id}/offline` - 下架文章
- `DELETE /api/content/article/{id}` - 删除文章
- `POST /api/content/article/{id}/like` - 点赞文章
- `POST /api/content/article/{id}/unlike` - 取消点赞

#### 统计分析服务（statistics-service）
- `POST /api/statistics/event` - 记录事件日志
- `GET /api/statistics/visit` - 获取PV/UV统计
- `GET /api/statistics/ranking` - 获取内容排行
- `GET /api/statistics/user-profile/{userId}` - 获取用户画像

#### 审计日志服务（audit-service）
- 自动记录所有Controller操作（AOP切面）

### 前端页面

#### admin-web（后台管理）
- `/login` - 登录页
- `/dashboard` - 首页
- `/system/menu` - 菜单管理
- `/system/role` - 角色管理
- `/system/user` - 账号管理

#### user-web（用户端）
- `/` - 首页（文章列表）
- `/article/:id` - 文章详情
- `/article/publish` - 发布文章
- `/topic` - 专题列表
- `/topic/:id` - 专题详情
- `/boiling-point` - 沸点动态
- `/login` - 登录页

## 🌐 多语言支持

### user-web多语言
- ✅ 中文简体（zh-CN）
- ✅ 英文（en-US）
- ✅ 语言切换功能（Header右上角）

## 🚀 启动方式

### 后端服务
```bash
# 启动所有服务
./start.sh

# 或单独启动
cd content-service && mvn spring-boot:run
cd audit-service && mvn spring-boot:run
cd statistics-service && mvn spring-boot:run
```

### 前端项目

#### 后台管理系统
```bash
cd admin-web
yarn install
yarn dev
# 访问 http://localhost:3000
```

#### 用户端前台
```bash
cd user-web
yarn install
yarn dev
# 访问 http://localhost:3001
```
