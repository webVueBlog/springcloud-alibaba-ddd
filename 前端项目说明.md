# 前端项目说明

## 项目结构

### 1. admin-web（后台管理系统）
**位置**：`admin-web/`
**端口**：3000
**用途**：系统管理员使用的后台管理界面

**功能模块**：
- ✅ 登录页面
- ✅ 首页Dashboard
- ✅ 菜单管理（树形结构、CRUD）
- ✅ 角色管理（权限分配）
- ✅ 账号管理（用户CRUD、重置密码、启用/禁用）

**技术栈**：
- Vue 3 + TypeScript + Vite
- Element Plus UI组件库
- Pinia状态管理
- Vue Router路由管理

### 2. user-web（用户端前台）
**位置**：`user-web/`
**端口**：3001
**用途**：普通用户使用的内容前台界面

**功能模块**：
- ✅ 首页（文章列表）
- ✅ 文章详情页
- ✅ 文章发布页（Markdown编辑器）
- ✅ 专题列表/详情
- ✅ 沸点动态（短内容发布）
- ✅ 用户登录
- ✅ 多语言支持（中文/英文）

**技术栈**：
- Vue 3 + TypeScript + Vite
- Element Plus UI组件库
- Pinia状态管理
- Vue Router路由管理
- Vue I18n多语言支持

## 启动方式

### 后台管理系统（admin-web）
```bash
cd admin-web
yarn install
yarn dev
# 访问 http://localhost:3000
```

### 用户端前台（user-web）
```bash
cd user-web
yarn install
yarn dev
# 访问 http://localhost:3001
```

## API接口说明

### 后台管理API
- 菜单管理：`/api/user/menu/**`
- 角色管理：`/api/user/role/**`
- 账号管理：`/api/user/user/**`

### 用户端API
- 文章：`/api/content/article/**`
- 专题：`/api/content/topic/**`
- 沸点：`/api/content/boiling-point/**`
- 认证：`/api/auth/**`

## 多语言支持

### user-web多语言
- 中文简体：`src/locales/zh-CN.ts`
- 英文：`src/locales/en-US.ts`
- 语言切换：通过Header右上角下拉选择

### 使用方式
```vue
<template>
  <div>{{ $t('common.home') }}</div>
</template>

<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
console.log(t('common.home'))
</script>
```

## 页面路由

### admin-web路由
- `/login` - 登录页
- `/dashboard` - 首页
- `/system/menu` - 菜单管理
- `/system/role` - 角色管理
- `/system/user` - 账号管理

### user-web路由
- `/` - 首页（文章列表）
- `/article/:id` - 文章详情
- `/article/publish` - 发布文章
- `/topic` - 专题列表
- `/topic/:id` - 专题详情
- `/boiling-point` - 沸点动态
- `/login` - 登录页

## 注意事项

1. **前后端分离**：前端项目独立运行，通过API与后端通信
2. **代理配置**：开发环境通过Vite代理转发API请求到后端
3. **认证机制**：使用JWT Token，存储在localStorage
4. **多语言**：user-web支持中英文切换，admin-web暂未实现
5. **响应式设计**：两个项目都支持PC端和移动端适配
