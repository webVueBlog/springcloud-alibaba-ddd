# User Web

用户端前端项目 - 技术社区内容前台

## 功能特性

- ✅ 文章列表/详情
- ✅ 文章发布
- ✅ 专题列表/详情
- ✅ 沸点动态
- ✅ 多语言支持（中文/英文）
- ✅ 用户登录

## 技术栈

- Vue 3
- TypeScript
- Vite
- Element Plus
- Pinia
- Vue Router
- Vue I18n

## 开发

```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn dev

# 构建生产版本
yarn build
```

## 项目结构

```
user-web/
├── src/
│   ├── api/          # API接口
│   ├── views/        # 页面组件
│   ├── router/       # 路由配置
│   ├── store/        # 状态管理
│   ├── locales/      # 多语言文件
│   └── main.ts       # 入口文件
```
