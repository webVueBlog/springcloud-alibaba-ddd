## 🚀 快速开始

### 1. 初始化数据库
```bash
# 执行SQL脚本
mysql -u root -p < sql/ddd_auth.sql
mysql -u root -p < sql/ddd_content.sql
mysql -u root -p < sql/ddd_statistics.sql
mysql -u root -p < sql/ddd_audit.sql
mysql -u root -p < sql/ddd_user_extend.sql
```

### 2. 启动后端服务
```bash
# 启动所有服务
./start.sh

# 或单独启动
cd content-service && mvn spring-boot:run
cd audit-service && mvn spring-boot:run
cd statistics-service && mvn spring-boot:run
```

### 3. 启动前端项目

#### 后台管理系统
```bash
cd admin-web
yarn install
yarn dev
# 访问 http://localhost:3000
```

#### 用户端前台
```bash
cd user-web
yarn install
yarn dev
# 访问 http://localhost:3001
```
